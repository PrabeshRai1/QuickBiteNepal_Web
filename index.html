About Project

QuickBite Nepal is a food ordering web application developed as a Final Year Project. It allows users to browse restaurants, order food online, and make secure payments.

Features

  • User Registration & Login
  • Restaurant Listing
  • Menu Management
  • Cart and Checkout System
  • Online Payment (eSewa, Khalti)
  • Order Tracking

Technologies Used

  • Frontend: React js
  • Backend: Laravel
  • Database: MySQL
  • Payment: eSewa, Khalti
